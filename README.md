# Wallet Demo Project
A Spring Boot-based wallet imitation application for user registration, money transfers, and more.

# Requirements
- Springboot 3.4
- Java 17
- Maven 3.2
- PostgresSQL

## Features
- User Registration
- Login
- Transfer Money
- Secure Password Encryption
- Validations

## Dependencies
- Spring Web
- PostgresSQL Driver
- Spring Data Jpa
- Validations
